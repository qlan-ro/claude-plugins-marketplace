# Hello Command

A simple greeting command to verify your plugin is working.

## Usage

```
/example-utils:hello
```

## Instructions

When the user runs this command:

1. Greet them warmly
2. Confirm the example-utils plugin is installed and working
3. List available commands in this plugin:
   - `/example-utils:hello` - This greeting command
   - `/example-utils:summarize` - Summarize code or files
   - `/example-utils:explain` - Explain code with examples

Keep the response friendly and concise.

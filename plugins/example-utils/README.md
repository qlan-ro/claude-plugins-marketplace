# Example Utils Plugin

A collection of utility commands and helpers for common development tasks.

## Installation

```bash
claude plugins:install github:YOUR_USERNAME/claude-plugins-marketplace/plugins/example-utils
```

## Commands

### `/example-utils:hello`
A simple greeting command to verify the plugin is working.

### `/example-utils:summarize`
Summarize the current file or selected code.

### `/example-utils:explain`
Get a detailed explanation of code with examples.

## Agents

### `code-reviewer`
A specialized agent for thorough code reviews.

## Skills

### `quick-docs`
Automatically generates documentation for functions and classes.

## Usage Examples

```
/example-utils:hello
/example-utils:summarize
/example-utils:explain
```

## Configuration

No additional configuration required.

# AGENTS.md Template

This template creates the root `AGENTS.md` file that points AI agents to project instructions.

## Purpose

The `AGENTS.md` file is a standard convention recognized by AI coding assistants. It tells agents where to find project-specific instructions.

## Template


## Usage

Generate this file at the project root alongside `CLAUDE.md` during the AI Workflow Configuration phase.
